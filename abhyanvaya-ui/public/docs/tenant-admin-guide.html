Setup catalog Roles & permissions

Guide for college administrators

This page explains how users, login types, and roles & permissions work in your college’s Abhyanvaya portal. You manage these under Setup → Roles & permissions (only college administrators can open this screen).

After you change roles or permissions for someone, ask them to sign out and sign in again. Their access is carried in their login session; a fresh sign-in applies your changes.

Two ideas: login type vs application roles

  • Login type (sometimes called enum role) is set when the account is created: typically Admin or Faculty. It defines broad behaviour—for example, only Admins can manage certain catalog screens; Faculty accounts used for teaching are often linked to a person in your Staff directory.
  • Application roles are labels you define for your college (for example “Exam officer”, “HoD”). Each role bundles a set of permissions (view students, manage courses, manage staff setup, and so on).

You assign one or more application roles to each user on the User assignments tab. If a user has no application roles, the system falls back to sensible defaults for their login type. As soon as you assign any application role to someone, their permissions come only from those roles—so make sure each role you assign includes everything they should be allowed to do.

Getting started with Roles & permissions

  1. Open Setup from the main menu, then Roles & permissions.
  2. On Application roles, create roles your college needs and open each role to tick the permissions that role should grant.
  3. On User assignments, pick a user and assign the application roles they should have. Save—then remind them to sign out and back in.

Faculty logins and Staff

When you add a Faculty user, you choose a staff profile from your Staff directory. That link tells the system which subjects that teacher may work with (based on subject assignments on the Staff record). If someone teaches new subjects later, update their assignments on the Staff page as usual.

You can change which staff profile is linked to a Faculty account from the same User assignments dialog when editing that user.

Passwords

  • Use Reset password on the User assignments tab to set a temporary password; the user will be asked to choose a new password after signing in.
  • Users can change their own password from the app header when logged in.
